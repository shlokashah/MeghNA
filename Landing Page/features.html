FEATURES


MeghNA provides the following functionalities and features relating to Cloud Analytics:


  • Cloud Detection: Given a TIR/VIS satellite image capture, clouds can be identified and pinpointed from the images and demarcated as per requirements.

  • Cloud Motion Prediction: A series of satellite images upto a particular timestamp can be used to nowcast the clouds and predict the future direction and motion. Forecasted cloud position images are returned accordingly.

  • Cloud Classification: Detected clouds are further classified into various types depending on temperature and height and such factors.

  • Cloud Attributes: Based on TIR1 & VIS1 counts, a lot of additional information relating to cloud attributes are returned, such as location, cloud top temp., cloud top height, etc.

  • Analytical Offerings: Cloud Analytics Report and Public APIs.
